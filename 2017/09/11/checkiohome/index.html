Non-unique Elements

问题描述

给出一个含有整数（X）的非空列表。
返回在此列表中的非唯一元素的列表。
不能改变列表的顺序。

问题分析

采用双重循环，从前到后，找到重复的元素,把重复元素利用javascript的push函数，push到一个新数组中。

问题描述

给出一个整数列表
返回列表的中位数（在列表有序的前提下，如果列表长度为奇数则，列表中间那个数据就是列表数据的中位数，如果列表数据的个数是偶数，则列表中间那2个数据的算术平均值就是列表数据的中位数）

问题分析

利用JavaScript的排序函数进行排序，再找出中位数。
注意： sort函数的参数必须是一个函数，如果不写参数，排序则按照ascii编码排序，在排序时特别注意如果不是按照ascii编码排序，要写排序函数。
